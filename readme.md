# F-Stroke Compiler
## Quick Start
### Build
```bash
./gradlew jar
```

### Run
```bash
java -jar compiler_project-1.0-SNAPSHOT.jar <filename>.fs <output> (optional)
```

### Test
```bash
./gradlew test
```
